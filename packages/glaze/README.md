# glaze

Header-only Glaze 7.2.0 package required by Hyprland's `hyprpm` and newer
HyprWM applications.

## Source provenance

- Official upstream tag: `stephenberry/glaze` `v7.2.0`
- No COPR, personal mirror, or downstream repack is used.

Developer mode, examples, fuzzers, optional network/TLS support, and EETF are
disabled. The package installs only public headers and the upstream CMake
package.

# hyprwm-fedora-release-local

Bootstrap package for the unsigned local filesystem repository. It installs a DNF5
configuration pointing at:

```text
/srv/hyprwm-fedora/fedora/44/current/$basearch/
```

The local-development configuration is intentionally unsigned. Public release
snapshots use the signing-aware repository configuration generated by
`tooling/hyprwm-packaging snapshot`, with both package and metadata signature
verification enabled.
